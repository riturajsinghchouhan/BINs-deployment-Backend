# Smart Dustbin Project

This repository contains both the Frontend and Backend for the Smart Dustbin application.

## Project Structure

- **UI/**: Frontend application (React + Vite)
- **server/**: Backend API (Node.js + Express + MongoDB)

## Getting Started

### Backend
1. Navigate to server: `cd server`
2. Install dependencies: `npm install`
3. Configure `.env` (see `server/README.md`)
4. Start server: `npm run dev`

### Frontend
1. Navigate to UI: `cd UI`
2. Install dependencies: `npm install` (if not already installed)
3. Start development server: `npm run dev`

## Features
- **Landing Page**: Animated hero section and improved UI.
- **Worker Management**: Admin dashboard for managing workers.
- **Real-time Monitoring**: Dustbin fill levels and status tracking.
- **Authentication**: Secure JWT-based login for Admins and Workers.
